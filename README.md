# Simple List

A simple application that CRUDs list items.

## Installation

```bash
$ yarn install
```

## Running the app

### Migrate & seed the database

```bash
# migrate
$ yarn run migration:run

# seed
$ yarn run seeder:run
```

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
